# The English Archive 📚
**Reclaiming the Margins of the Literary Canon**

An anonymous, open-access digital repository providing high-quality, curated analysis for BA English Honours students. This platform is designed to provide affordable, peer-reviewed scholarship for those who lack access to expensive academic resources.

---

## ✨ Features
- **Magazine-Style Interface:** Browse manuscripts through a clean, grid-based gallery.
- **Deep Search:** Instant filtering by Title, Syllabus/Paper Code, or Keyword.
- **Rich Text Analysis:** Scholarly notes presented in elegant, readable typography.
- **Secure Editorial Collective:** A private management portal for curating and committing manuscripts.
- **Anonymous Peer Feedback:** A direct, private channel for student queries and critiques.

## 🛠️ Technology Stack
- **Backend:** Python & SQLite
- **Frontend:** Streamlit & Custom CSS
- **Editor:** Quill (Rich Text formatting)
